Proyecto: Agroecoalmacen - Módulo Gestión de Plantas
Evidencia: GA7-220501096-AA2-EV01
Estudiante: Rafael Hernandez

Descripción:
Este módulo implementa el CRUD de plantas utilizando Java y JDBC, conectado a una base de datos MySQL. 
Hace parte del proyecto Agroecoalmacen

Funcionalidades incluidas:
 Insertar plantas
 Listar plantas
 Buscar por ID
 Actualizar plantas
 Eliminar plantas

Tecnologías:
- Java
- JDBC
- MySQL

Instrucciones:
1. Importar el archivo SQL en MySQL.
2. Abrir el proyecto en NetBeans.
3. Ejecutar la clase Main.java para probar el módulo.

Estructura del proyecto:
- Planta.java → Modelo
- PlantaDAO.java → CRUD
- ConexionBD.java → Conexión JDBC
- Main.java → Pruebas
